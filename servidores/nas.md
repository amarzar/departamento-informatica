Configuración antigua de Bender
```
ip dhcp pool servidor_NAS
 host 172.16.1.3 255.255.0.0
 hardware-address 0050.4377.09e0
 client-name nas.servidor
 default-router 172.16.0.1 
 dns-server 80.58.61.250 172.16.0.1 
```
